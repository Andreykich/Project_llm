## Метрики оценки качества и пайплайн валидации

Для демонстрации качества вероятностной LLM-агентной системы используется воспроизводимый пайплайн оценки.

**Набор тестов:** 50 тест-кейсов, стратифицированных по 10 доменам (по 5 кейсов на домен). Каждый кейс содержит промпт и чек-лист обязательных секций ответа.

**Пайплайн прогона:** для каждого кейса сохраняются артефакты всех агентов (вход/выход) и метаданные (latency, статус ошибок). Сырые логи позволяют полностью воспроизвести результаты и проверить корректность расчёта метрик.

**Автоматические метрики:**
- `pass_rate` по статусу выполнения пайплайна без ошибок;
- `latency` (p50/p90/p95);
- `sections_coverage` — покрытие обязательных секций ответа;
- `sources_rate` — доля ответов с >=2 источниками/ссылками.

**Экспертные метрики (rubric 0–5):**
Relevance, Specificity, Actionability, Evidence. Разметка проводится двумя оценщиками; итог — среднее значение и доля кейсов, прошедших порог PASS.

Полный датасет тест-кейсов и скрипты воспроизводимой оценки приложены в папке `/eval` репозитория.
